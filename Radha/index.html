Example: From Phrase to SQL Query

Input Phrase:
Show total sales and customer count by department in descending order.

Generated SQL: 

SELECT department,
       SUM(sales_amount) AS total_sales,
       COUNT(DISTINCT customer_id) AS customer_count
FROM sales_data
GROUP BY department
ORDER BY total_sales DESC;
